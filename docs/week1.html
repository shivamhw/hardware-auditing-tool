Objective (4/12/21)

create a web server hosted on Pi that can control a power supply.

Justification

to perform voltage glitching, essentially allowing pi to turn off or on the target device to make target device boot into a unsafe state.

Steps :

  1. Making circuit based on lm317( Variable power supply)
  2. Making the controlling interface on pi
  3. Exposing that control on a webpage hosted by pi
